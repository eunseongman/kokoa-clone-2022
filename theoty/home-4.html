position: fixed - 박스가 초기에 위치한 화면 그대로 유지
(but, top / bottom / left / right 등의 프로퍼티를 사용하면 box들을 무시 > 가장 상위 레이어)
position: static - 기본값, 초기 위치 그대로 둠
★ position: relative - top. left ... 의 프로퍼티를 통해 위치 미세조정
★ position: absolute - relative 한 가장 부모를 찾아 해당 부모의 위치에 조정함

combinators -
p span {} : p 안에 있는 span에 적용
div > span {} : div 바로 밑 자식에게 적용
p + sapn {} : p 바로 다음 형제 span에 적용
p ~ span {} : p 바로 다음에 오지 않더라도 적용
input[placeholder="name"] {} : attribute를 검사하여 적용
( ~= / *= / ^= / $= ... 여러 옵션 존재)

pseudo selector - 세부적으로 엘리먼트를 선택해주는 것,,,
(ex - div:first-child {}, span:nth-child(2n+1), )
+ attribute를 지정하여 사용할 수 있음 -
input:required {}
+ 점을 두번 쓰는 경우 '::'
input::placeholder (미리보기 글자 변경), p::selection(글자 드래그),
p::first-letter(첫 글자), p::first-line(첫 줄) ...

state -
active : 해당 요소를 마우스로 클릭중인 상태
hover : 해당 요소에 마우스 커서를 갖다댄 상태
focus : 해당 요소를 키보드로 선택중인 상태
visited : 방문했었던 웹사이트인 상태 (링크에만 적용)
focus-within : focused인 자식을 가진 부모 엘리먼트의 상태
(+ combinator와 state를 섞은 형태를 자주 사용함 ex- form:hover input:focus {} )

color - 사용자 지정 컬러 , 16진수 #D7BFD7, rgb & rgba(투명도) rgba(252, 206, 0, 0.5),

: root { --variable-name : value; } - custom property (변수) 저장
color : var(--variable-name); - custom property (변수) 사용
